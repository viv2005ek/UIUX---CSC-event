# UIUX
PROBLEM STATEMENT : 6. Infectious Disease Awareness & Visualization Platform
Problem Context:
Public health data on infectious diseases is often complex and can cause panic or 
misinformation if poorly presented.
Design Objective:
Design a clear, trustworthy interface visualizing state-wise disease spread across India, 
trends over time, and basic educational context without inducing alarm.
